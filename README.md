## About The Project
Our project takes inspiration from the Pokemon franchise's "Pokedex", a pocket encyclopedia that the player or Pokemon Trainer fills out as they encounter, meet, and befriend new Pokemon. One interesting spin on the Pokedex was seen in the game *Pokemon Legends: Arceus*. The game's setting takes place during the earliest point in the franchise's timeline, during a period reminiscent of feudal Japan. In this game, filling out the Pokedex is not as easy as in other games, as your player is recording information by hand, with no electronic assistance. This means that in order to fill out your Pokedex, the player must encounter the Pokemon multiple times, in multiple different conditions. This enhances the experience of the game, as the player becomes more familiar with the Pokemon as you are more exposed to them.

And thus, the idea was formed: "Why not make a system that helps to facilitate that between people?" 

FrienDex has been designed to work fully offline and does not rely on an online database. All of your information remains on your phone.

### About the name

FrienDeX, FrienDex, and Friendex were named after it's inspiration, the PokeDex. The reason that the F, D, and X are boldened in the title of this README is because FrienDeX can also be short for "**Friend**ship **D**atabase **eX**plorer"! It's a little unconventional, but if Capcom can make the "*Pe*rsonal *T*erminal", or "PeT", then we can keep our acronym, too.

## Using the FrienDex App

### FrienDex Rolodex functions:
1. **Add Dex Entry allows** for the creation of new dex entries of the people you meet, it contains text boxes for first and last names as well as a toggle for the favorites indicator.
2. Once an entry has been created, the listing can be selected to view Rooms, Data Blocks, and the Editing and Delete functions.
3. Delete functions as stated, allowing you to delete a listing permanently.
4. The edit feature allows not just for the changing of the name, but also the ability to add blocks.
5. Blocks can be filled with a wide variety of data so that information can easily be saved about the various individuals in the dex, you simply select a type and fill out the boxes that appear. Once done, select **Add Block**. 


### FrienDex Room functions:
1. Rooms allow for the various individuals added on the rolodex side to be more easily sorted into categories, you can add your first room with the Add Room button where a name and short description is required before you can press save and create the room.
2. Once a room has been created, the listing can be selected to view details and members. 
3. Additional members can be added with the Add Members button by selecting created individuals from the list and selecting Add Members.
4. Members can be removed with the Remove Members button by selecting participating individuals from the list and selecting Remove Members.
